# 底部导航栏图标颜色方案说明

## 当前配色方案

### 方案一：红色 → 金色（推荐）✨

**未选中状态**：
- 图标颜色：**红色 (#C80000)**
- 文字颜色：**红色 (#C80000)**

**选中状态**：
- 图标颜色：**金色 (#D4AF37)**
- 文字颜色：**金色 (#D4AF37)**

**优点**：
- ✅ 红色+金色是经典搭配（国旗色）
- ✅ 对比度明显，用户容易识别选中状态
- ✅ 符合红色文化主题的庄严感
- ✅ 金色在红色背景上视觉冲击力强

**设计建议**：
- 未选中图标：使用红色 (#C80000) 填充或描边
- 选中图标：使用金色 (#D4AF37) 填充，可以加粗或加光效

## 图标设计规范

### 图标尺寸
- **尺寸**：81x81px（微信小程序tabBar标准尺寸）
- **格式**：PNG（支持透明背景）
- **大小**：建议 < 40KB

### 设计要点

1. **未选中状态**：
   - 建议使用描边样式（线条图标）
   - 或者使用半透明填充
   - 线条粗细：2-3px
   - 保持简洁清晰

2. **选中状态**：
   - 使用填充样式（实心图标）
   - 可以加粗或增加光效
   - 线条粗细：4-5px
   - 增强视觉冲击力

3. **统一风格**：
   - 所有图标使用相同的设计风格
   - 线条粗细、圆角、风格保持一致
   - 建议使用线性图标或扁平化图标

### 图标示例（建议）

```
未选中（红色描边）         选中（金色填充）
   ┌─────┐                  ┌─────┐
   │  ⭕ │                  │  ⭐ │
   └─────┘                  └─────┘
   红色线条                 金色填充
   #C80000                 #D4AF37
```

## 推荐方案总结

### ⭐ 首选：红色 → 金色

**配置**：
```json
{
  "color": "#C80000",        // 未选中：红色
  "selectedColor": "#D4AF37"  // 选中：金色
}
```

**原因**：
- 符合红色文化主题
- 金色代表荣誉和庄严
- 视觉对比强烈
- 用户体验好

### 备选：浅红 → 深红

**配置**：
```json
{
  "color": "#FF6B6B",        // 未选中：浅红色
  "selectedColor": "#C80000" // 选中：深红色
}
```

**原因**：
- 统一红色系
- 渐变自然
- 符合主题一致性

## 注意事项

⚠️ **重要提示**：
- 确保图标在两种状态下都能清晰识别
- 选中状态应该比未选中状态更突出
- 保持所有图标的视觉权重一致
- 测试在不同设备上的显示效果
